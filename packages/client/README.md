# Openboard Client

Frontend application for Openboard.

## Tech Stack

- React
- Vite
- TypeScript
- React Router DOM
- @dnd-kit (Drag and Drop)
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## Scripts

- `dev`: Start development server
- `build`: Build for production
- `preview`: Preview production build
